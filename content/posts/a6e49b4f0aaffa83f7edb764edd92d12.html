Tor项目近日发布推文号召网友捐款：

🧅 Tor是构建自由互联网的基石，守护您的隐私安全。作为非营利组织，我们依靠捐赠来支持数百万用户信赖的工具。这些工具始终免费使用，绝不为牟利而收集、出售、交易或出租您的任何数据。

立即捐款，助力Tor持续发展，共筑更美好的互联网未来。所有捐款都将获得我们的支持者“Power Up Privacy”计划的等额匹配。这意味着25美元的捐款将产生50美元的实际价值。

Tails及Tor浏览器的新版本发布

Tails推出7.1版本

Tails操作系统的7.1版本将Tails中的Tor浏览器主页更改为离线页面，与Tails之外的Tor浏览器主页非常相似，而不是访问Tor官方网站的在线页面。同时改进了当打开应用程序需要管理密码但欢迎屏幕中未设置管理密码时的弹出消息。

此外，还做了如下更新：

  • 将Tor浏览器更新至14.5.8。
  • 将Tor客户端更新至 0.4.8.19。
  • 将Thunderbird更新至140.3.0。
  • 移除ifupdown安装包。
  • 在Tor浏览器的新标签页中隐藏“您的Tor连接不受Tor浏览器管理”的提示消息。( #21215 )

Tor浏览器推出14.5.8及14.5.9两次版本更新

Tor 浏览器 14.5.7以来的完整更新日志如下：

  • 所有平台
    • Tor 已更新至 0.4.8.19
    • OpenSSL 已更新至 3.5.4
    • Bug tor-browser#44239：使用最安全的安全设置时，DDG HTML 页面和搜索结果显示不正确
    • Bug tor-browser#44240：dom.security.https_first_add_exception_on_failure 上的拼写错误
    • Bug tor-browser#44244：从 Firefox 144 反向移植安全修复
    • Bug tor-browser-build#41574：更新 Snowflake 内置桥接线路
  • Windows + macOS + Linux
    • Bug tor-browser#44032：为桌面稳定版实施 YEC 2025 Takeover
  • 安卓
    • Bug tor-browser#44031：为 Android Stable 版本实施 YEC 2025 接管
    • Bug tor-browser#44263：YEC 无法在 Tor 浏览器 14.5.8 Android 上显示（14.5.9版本更新）

Tor项目2025年度募捐活动

Tor发布公告称，Tor是自由互联网的基石，需要大家共同努力来改进和扩展它。Tor项目将在接下来的三个月里举办一场筹款活动，号召大家支持，共同推进数字自由。

Tor项目称其与令人难以置信的支持者社区一起，不仅在想象一个自由的互联网，而且正在构建一个自由的互联网：

  • 在土库曼斯坦等信息访问受到严格控制的国家，Turkmen.news相信Tor能够提供曾经不存在的免费互联网的安全访问。
  • 新闻自由基金会使用Tor作为SecureDrop的支柱，该工具允许记者和消息来源自由交流，而不必担心遭到报复或曝光。
  • 在俄罗斯和埃及，当局经常封锁加密工具来控制通信，个人使用Tor浏览器访问Tuta Mail等加密电子邮件服务以自由通信。

Tor项目表示，作为一家 501(c)3 非营利组织，Tor项目依靠捐款来支持其工具——Tor网络、Tor浏览器、Tails操作系统和Tor生态系统。这些工具深受数百万用户的信赖，并且始终免费使用，不会收集、出售、交易或出租用户的任何数据以牟利。

Tor项目呼吁大家继续支持Tor，让互联网自由起来。如果在活动期间捐款，Power Up Privacy的支持者将为捐赠者匹配等额捐款，也就是25美元的捐款将产生50美元的影响力，且所有超过25美元的捐款都有资格获得Tor提供的趣味周边商品。

Tor强调，在2023-2024财年，其84%的支出与项目服务相关，意味着其预算中很大一部分直接用于Tor的建设和改进。这些项目服务包括构建Tor技术、开展推广活动以及维护Tor工具的更新和安全。Tor获得了慈善导航（Charity Navigator）的四星级慈善评级，并荣获Candid的透明度白金印章。这表明Tor项目致力于以公开透明的方式管理财务，并利用用户的投资捐赠实现更大的影响力。
